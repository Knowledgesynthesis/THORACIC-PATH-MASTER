# **7. THORACIC PATHOLOGY REASONING FRAMEWORK**

1. Identify anatomic compartment (lung, mediastinum, pleura, airway).  
2. Architecture first (glandular, squamous, NE, mesothelial, spindle).  
3. Cytology + stromal context.  
4. Evaluate pattern-specific features (fibrosis type, nest morphology, cell type).  
5. Apply IHC conceptually.  
6. Add molecular context (high-level only).  
7. Integrate into a broad diagnostic family.  
8. Identify pitfalls & mimics.

Pitfalls:  
- Misinterpreting reactive pneumocytes as adenocarcinoma.  
- Confusing carcinoid with SCLC.  
- Calling atypical mesothelial proliferation mesothelioma.  
- Overcalling thymoma vs lymphoma.
